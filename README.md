# Kanapka Theme

Lightweight modular WordPress and WooCommerce theme for kanapka.kiev.ua.

## Principles

- WordPress and WooCommerce APIs are the primary data source.
- Page and component assets load only when required.
- Repeated UI is implemented as reusable template parts.
- SEO metadata stays under WordPress, WooCommerce and the active SEO plugin.
- Secure Custom Fields integration is optional and limited to data unavailable through core APIs.
